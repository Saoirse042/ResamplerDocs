    OpenUTAU expressions

    OpenUTAU uses expressions to manage flags and other arguments that renderers may use.
    Information is taken directly from the OpenUTAU wiki here, where you can also learn about setting custom expressions.

    Flag Name Abbreviation Range Default Purpose
    Velocity VEL 0 - 200 100 Consonant velocity (stretch/shorten preutterance)
    Volume VOL 0 - 200 100 Note volume
    Attack ATK 0 - 200 100 Envelope starting volume
    Decay DEC 0 - 100 0 Envelope ending volume
    Voice color CLR Utilizes voicebank appends/colors
    Resampler engine ENG worldline64.exe Change resamplers on the fly
    g Gender GEN -100 - 100 0 Formant shift
    B Breath BRE 0 - 100 0 Breathiness
    H Lowpass LPF 0 - 100 0 Low pass filter
    Modulation MOD 0 - 100 0 Note modulation from original pitch
    Alternate ALT 0 - 16 0 Alternate aliasing for duplicates in oto.ini
    Tone shift SHFT -36 - 36 0 Allows for selecting pitch alias

    Hey! Where's my Moresampler flags gone? They were in the project I was working on last week!

    Custom expressions are only saved per project! If you have expressions or flags for a renderer you use frequently, you should create a template project to load for each session.
